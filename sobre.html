Sobre o Projeto

Este simulador foi criado com fins educativos, para demonstrar de forma prática e intuitiva como ocorrem ataques comuns na área de Segurança da Informação.

🎯 Objetivo

Auxiliar estudantes e curiosos da área a entenderem visualmente como ocorrem alguns dos ataques cibernéticos mais frequentes, sem colocar em risco sistemas reais.

🔍 Ataques Simulados

  • Phishing: Simula um site falso que coleta e-mails para mostrar como dados podem ser capturados.
  • Ransomware: Demonstra a criptografia de arquivos e a exigência de “resgate”.
  • SQL Injection: Mostra como comandos maliciosos podem invadir bancos de dados via campos de login.
  • Brute Force: Tenta descobrir a senha digitada simulando força bruta realista.

📌 Tecnologias Utilizadas

HTML5, CSS3 e JavaScript puro. O projeto é leve, acessível e 100% responsivo.
